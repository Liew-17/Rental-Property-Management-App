## Overview

This project is a **mobile application for rental property management**, designed to help landlords and property managers efficiently handle rental properties, tenants, and automated tasks. The system includes a backend server, a Flutter-based frontend app, and an AI module for predictive analytics related to property management.  

---

## System Structure

The project is organized into three main directories:

1. **Backend**  
   Contains the server-side code for handling requests, managing database operations, and automating tasks.

2. **Flutter Application**  
   Contains the mobile app frontend built with Flutter.

3. **AI_Training**  
   Contains code for gathering data, cleaning it, and training AI models for predictive functionality.  

---

## Prerequisites

Before running the system, ensure you have the following installed:

- Python 3.x  
- Flutter SDK  
- Required Python packages (install via `pip install -r requirements.txt`)  
- A connected device or emulator for Flutter
